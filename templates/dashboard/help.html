❓ Aide & FAQ

Tout ce que vous devez savoir pour utiliser Joe Dev Academy

Comment fonctionne l’abonnement ? +
Vous choisissez un plan mensuel, vous payez, puis vous accédez à toutes les formations pendant la durée de votre abonnement.
Pourquoi je n’arrive pas à accéder aux cours ? +
Vous devez être connecté et avoir un abonnement actif. Si votre abonnement a expiré, l'accès est bloqué automatiquement.
Où trouver mes cours ? +
Tous vos cours sont disponibles dans votre Dashboard après connexion.
Comment payer mon abonnement ? +
Paiement via Mobile Money. Ensuite vous envoyez la preuve pour validation.
Que se passe-t-il si mon abonnement expire ? +
L’accès aux cours est bloqué automatiquement jusqu’au renouvellement.
Comment renouveler ? +
Retournez sur la page abonnement et effectuez un nouveau paiement.
📞 Comment contacter le support ? +
Contactez-nous via WhatsApp ou la page Contact.
